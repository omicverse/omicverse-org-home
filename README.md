# OmicVerse.org Home

Static community homepage for `omicverse.org`.

The site positions OmicVerse as an open multi-omics community focused on:

- unified Python APIs for omics analysis,
- Python/Rust re-implementations of widely used R algorithms,
- interoperable data foundations,
- tutorials, benchmarks, and agent-ready workflows.

## Development

```bash
npm run build
npm run serve
```

Then open:

```text
http://localhost:4173
```

## Structure

```text
src/
  index.html
  styles.css
  script.js
dist/
```

`dist/` is generated and can be deployed directly as a static site.
