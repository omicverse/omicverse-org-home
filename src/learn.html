Quickstart

Install OmicVerse, load a public PBMC dataset, run preprocessing and clustering, embed in 2D.

$ pip install omicverse

# in Python
import omicverse as ov
adata = ov.read("pbmc3k.h5ad")
ov.pp.preprocess(adata)
ov.tl.cluster(adata, method="leiden")
ov.pl.embedding(adata, color="leiden")
